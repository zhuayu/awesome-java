# 变量

## 变量的基本概念

变量就是申请内存来存储值。也就是说，当创建变量的时候，需要在内存中申请空间。内存管理系统根据变量的类型为变量分配存储空间，分配的空间只能用来储存该类型数据。

- 变量的声明方式：`数据类型 变量名 = 初始值 ` `example： int num = 8;`
- 变量的注意事项：
  - 变量在使用前必须声明其数据类型
  - 变量在使用前必须初始化
  - 变量不能重复声明
- 标识符命名规则
  - 由数字、字母、下划线及$等组成，==不能以数字开头==
  - 不可以使用java语言的关键字 
  - 区分大小写，不限制长度但不宜过长
  - 支持中文，不常用

# 运算符

## 关系运算符

- `>` 表示是否大于运算符

- `<` 表示是否小于运算符

- `>=` 表示是否大于等于运算符

- `<=` 表示是否小于等于运算符

- `==` 表示是否等于运算符

- `!=` 表示是否不等于运算符

==注意：所有的关系运算符最终运算的表达式的结果一定是布尔类型==

## 逻辑运算符

`&&` 逻辑与，相当于并且，逻辑运算符两边表达式同真为真，一假为假

`||` 逻辑或，相当于活着，逻辑运算符两边表达式一真为真，同假为假

`!` 逻辑非，相当于取反，真为假，假为真

逻辑运算符的短路特性：

- 对于逻辑与运算符来说，若第一个表达式为假，则结果为假，跳过第二个表达式。
- 对于逻辑或运算符来说，若第一个表达式为真，则结果为真，跳过第二个表达式。

## 条件/三目运算符

- 条件表达式？表达式1:表达式2
- 判断条件表达式是否成立，若成立则执行表达式1，否则执行表达式2。

## 移位运算符

- `<<` 左移运算符，用于将数据的二进制位向左移动，右边使用0补充，左移一位相当于扩大2倍。
- `>>` 右移运算符，用于将数据的二进制位向右移动，左边使用符号位补充（符号右移）。
- `>>>` 逻辑右移，用于将数据的二进制位向右移动，左边使用0补充。

## 位运算符

- `&` 按位与运算符，二进制位进行与运算，同1为1，一0为0
- `|` 按位或运算符，二进制位进行或运算，有1为1，同0为0
- `~` 按位取反，二进制位进行取反运算，1为0，0为1
- `^` 按位异或，二进制位进行异或运算，同为0，不同为1

# 流程控制语句

## 分支结构

### if分支结构

```java
if(expression){
  ...
}
```

- 判断expression是否成立
  - 成立，执行语句块的内容
  - 不成立，则跳过

### if else分支结构

```java
if(expression) {
  block1；
}else{
  block2；
}
```

- 判断expression是否成立
  - 成立，则执行block1；
  - 不成立，则执行block2；

### if else if else分支结构

```java
if(expression1){
  block1;
}else if(expression2){
  block2;
}else{
  block3;
}
```

- 判断expression1是否成立
  - 成立，则执行block1；
  - 不成立，则判断expression2是否成立
    - 成立，则执行block2
    - 不成立，则执行block3；

### swich case分支结构

```java
swich(变量/表达式){
  case 字面值1:block1；break；
  case 字面值2：block2；break；
  ...
  default: blockn;
}
```

- 计算变量/表达式的值，判断是否匹配字面值1
  - 若匹配，则执行block1，执行break跳出当前结构
  - 若不匹配，则判断是否匹配字面值2
    - ......

==swich()中支持的数据类型有：byte、short、char、int,从jdk1.5开始支持枚举，从jdk1.7开始支持String==

`注意：swich语句中，如果没有break语句，则会一直执行。`

## 循环结构

### for循环

```java
for(初始化表达式；条件表达式；修改初始值表达式){
  循环体；
}
```

- 执行初始化表达式，判断条件是否成立
  - 若成立，则执行循环体，并修改初始值表达式，判断条件是否成立
  - 若不成立，则循环结束

==continue：用于结束本次循环开始执行下一次循环==

==break；用于跳出循环体==

==for(;;):没有条件的循环，无限循环，又称死循环==

### break关键字跳出多层循环

- 在嵌套的循环结构中，break用于退出所在的循环体
- 如果想要退出外层循环体，需要使用标号的方式

```java
outer:for(...){
  for(...){
    break outer;
  }
}
```

### while 循环

```java
while(expression){
  循环体；
}
```

- 判断条件是否成立
  - 成立，则执行循环体
  - 不成立，则循环结束

==while(true):表示无限循环== 

### do while循环

```java
do{
  循环体；
}while(expression)
```

- 执行循环体，判断条件是否成立
  - 成立，则执行循环体，判断条件是否成立
  - 不成立，则循环结束

==do while循环主要用于至少执行一次循环体的场合中==

# 数组

## 一维数组

#### 一维数组的声明

```java
// 数据类型[] 数组名称 = new 数据类型[数组长度]
int[] arr = new int[10];
// 获取数组的长度，获取string的长度用s.length()；
arr.length;
// 基本类型的数组创建后，其元素的初始值，byte、short、char、int、long为0，float和double为0.0，boolean为false；
// 可以在数组声明的同时进行初始化
// 数据类型[] 数组名 = {value1,value2,...}
```

#### 数组工具类

```java
java.util.Arrays.xxx;
// 输出数组中的内容
static String toString(int[] a);
// 将参数指定元素赋值给数组中的所有元素
static void fill(int[] a,int val);
// 判断两个数组元素内容和次序书否相同
static boolean equals(boolean[] a,boolean[] b);
// 对数组元素从小到大排序
static void sort(int[] a);
//从数组中查找指定元素所在的位置
static int binarySearch(int[] a,int key);
```
